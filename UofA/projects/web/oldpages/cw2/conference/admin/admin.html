Initialize Poster:

This script will;

  • add a new poster room entry
  • create the poster directory
  • copy the template file to the new directory
  • modify the navBar.html and chatlogin.html file for the new poster
  • make a new chat server.conf file
  • restart chat (you must restart EwgieServer by hand)
  • create a news group for the poster (you must create the news group by hand)

This can all be done with just the poster title author list and email addresses the short title for the chat room and discussion group and a schedule time for the live chat. The actual poster GIF files can be uploaded later. When the poster GIF files are uploaded the navBar.html file will need to be trimed to the appropriate number of slides (easy).
