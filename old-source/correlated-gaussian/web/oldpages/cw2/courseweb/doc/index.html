CourseWeb - Class WebSite system

CourseWeb is a system of web pages and web-page management tools for course related websites. CourseWeb consists of two major components:

  • The Student Interface: allows the users to easily find course related web pages and navigate those web pages using a drop-down menu.

  • The Instructor Interface: allows instructors to easily build and modify the Student Interface for their course using web-page based tools.

CourseWeb provides the students and instructors with a consistent website navigation tool for each course offered by the department. Instructors have complete control over the website content.

CourseWeb also includes a web-based interface to create the course listings and initial course web-sites at the beginning of each semester. This Beginning Of Semester System is called B.O.S.S.

NOTES:

  • The current version of CourseWeb uses special features only available in Netscape 3.0 (or newer).
  • To avoid problems with Netscape page caching please make sure your Netscape is configured as follows: Options, Network Preferences, Cache tab, Verify Documents - Every Time.
