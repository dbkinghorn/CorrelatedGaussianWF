Easy Web Group Interaction Enabler

Concise Help


Enter/Exit Commands
/join name Joins the chat area with a nickname.
/quit Leaves the chat area.
/connect name host (port) Connects to another Ewgie server. If a port number is supplied, connects to another Ewgie server at the given port.

General Commands
/help Provides concise online help.
/who (name) Shows everyone that is online. If a name is supplied, shows information on somebody.
/whowas (name) (limit) Shows a specific person who has been in chat. If no name is supplied, shows the last 100 people who were in chat. If a numerical limit is supplied, shows the last limit people who have been in chat.
/nick name Renames yourself.
/away (text) Indicates that you are away. If no text is supplied, indicates that you are away with a default message.
/return Returns you from being away.

Conversation Commands
/msg name text Sends a private message to somebody.
/ring name Rings somebody.
/me text Describes an action.
/notify (name) Notifies you when someone enters and leaves chat. If no name is supplied, checks your notification list to see who is present.
/nonotify (name) Removes someone from your notification list. If no name is supplied, your notification list is cleared.
/ignore (name) Ignores private messages from somebody. If no name is supplied, ignores private messages from everybody.
/noignore (name) Does not ignore private messages from somebody. If no name is supplied, does not ignore private messages from everybody.
/filter name Ignores all messages from somebody.
/nofilter (name) Does not ignore all messages from somebody. If no name is supplied, does not ignore all messages from everybody.
/follow (name) Follows somebody. If no name is supplied, follows everybody.
/nofollow (name) Doesn't follow somebody. If no name is supplied, doesn't follow anybody.

Room Commands
/join room Joins a room. If the room does not exist a new one is created.
/leave (room) Leaves a room and places you in the server's main room. If no room name is supplied you will leave chat.
/remove room Removes a room. A room must be empty before it can be removed.
/topic (text) Changes the topic for the room you are in. If no text is supplied the topic is removed.
/room (room) Displays information about a room. If no room is supplied, displays information about the current room.
/rooms (text) (number) Displays all rooms and their topics. If text is supplied, displays only rooms and topics containing that text. If a number is supplied, displays only rooms with that number of people.
/names (text) (number) Displays all rooms and people. If text is supplied, displays only rooms and names containing that text. If a number is supplied, displays only rooms with that number of people.
/invite name Invites a person to the room you are in.

Other Commands
/fontsize size Sets the size of the chat font.
/motd Displays the message of the day.
/clear Clears the screen.
/stats Displays server information.
/client Displays client information.
 
Names must be one word. Names specified in commands are case-insensitive. Multiple names can be specified in a command using a comma-delimited list.

Room names must be at least one character. Room names and topics cannot have commas in them.
